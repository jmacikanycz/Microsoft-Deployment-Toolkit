Using the TrackPoint Center Button

  トラックポイントのセンター・ボタン

 


トラックポイントのセンター・ボタンを用いて、Webページや長い文書、大きなスプレッド・シート等を簡単にスクロールさせたり、あるいは、画面の一部分を拡大したりすることができます。


トラックポイントのスクロール機能を使う
トラックポイントの拡大表示機能を使う
センター・ボタンの機能を素早く切り替える
センター・ボタンの設定を変更する

 

 

 

 

トラックポイントのスクロール機能を使う

 

  1. スクロールさせたいWebページや文書の上にマウス・ポインターを移動します。

     

  2. トラックポイントのセンター・ボタンを押します。センター・ボタンが押されている間、マウス・ポインターはスクロールのアイコンに変わります。（センター・ボタンの機能としてスクロール機能が選択されている場合）

     

  3. Webページや文書をスクロールさせるには、トラックポイントのセンター・ボタンを押しながら、スクロールさせたい方向にスティックを押します。縦方向にも横方向にもスクロールさせることができます。

     

 

注： スクロールする速さは、トラックポイントのスティックを押す強さで決まります。速くスクロールさせるには、スティックを強く押してください。ゆっくりとスクロールさせるには、スティックを軽く押してください。また、スクロールの対象となるウィンドウは、マウス・ポインターの真下のウィンドウになります。スクロールさせたいウィンドウを画面の手前に移動させる必要はないため、画面上のウィンドウの配置を変えることなくスクロールさせることができます。

 

 

 

トラックポイントの拡大表示機能を使う

 

  1. 画面上の拡大表示させたい場所にマウス・ポインターを移動します。

     

  2. トラックポイントのセンター・ボタンを押すと、マウス・ポインターの周辺部分を拡大したウィンドウ表示されます。センター・ボタンが押されている間、マウス・ポインターは拡大表示アイコンに変わります。（センター・ボタンの機能として拡大表示機能が選択されている場合）

     

  3. 拡大表示ウィンドウを移動させるには、トラックポイントのセンター・ボタンを押しながら、移動させたい方向にスティックを押します。

     

 

注： 拡大表示ウィンドウを閉じるには、トラックポイントのセンター・ボタンをもう一度押します。拡大表示ウィンドウを動かしている間は、表示されている内容は維持されますが、約１５秒間何も操作がなされなかった場合は、拡大表示ウィンドウは自動的に閉じられます。

 

センター・ボタンを押しながら、左または右のトラックポイント・ボタンを押してポップアップ・メニューを表示させ、項目を選択すると、拡大表示ウィンドウの拡大率またはサイズを変更できます。拡大表示機能には、拡大表示ウィンドウ内のイメージをビットマップ形式で、クリップボードにコピーするオプションがあります。コピーするには、センター・ボタンを押しながら左ボタンまたは右ボタンを押し、ポップアップ・メニューの中から、「クリップ・ボードにコピーする」を選択します。「1x ズーム」 と 「ウィンドウサイズ - 任意」を組み合わせて使用すると、画面上の任意の部分を簡単にキャプチャーすることができるようになります。拡大表示ウィンドウのサイズを任意に変更するには、下のセンター・ボタンの設定を変更する を参照してください。

 

 

 

センター・ボタンの機能を素早く切り替える


 

タスクバーの通知領域に登録されたトラックポイントのアイコン（あるいは UltraNavのアイコン）のメニューから、トラックポイントのセンター・ボタンの機能を素早く切り替えることができます。

 

  1. タスクバーの通知領域に登録されたトラックポイントのアイコン（あるいは UltraNavのアイコン）をクリックするとメニューが開きます。

     

  2. センター・ボタンの機能を使用するには、（UltraNavのモデルの場合は、トラックポイントの設定を選択し）スクロールか拡大表示を選択します。オフを選択すると、これらの機能は使用不可になります。

 

注： メニューの中からスクロールを選択した場合は、スタンダード、スムーズ、自動選択の中から、スクロールのタイプを選択します。

 

 

 

センター・ボタンの設定を変更する


 

トラックポイントのセンター・ボタンの機能の設定は、トラックポイントのプロパティ・パネル（UltraNavのモデルの場合は UltraNavのプロパティ・パネル）で行います。また、これらのパネルは、マウスのプロパティ・パネルから開くことができます。

 

  1. コントロール パネル フォルダーから「マウスのプロパティ」を開きます。タスクバーの通知領域にトラックポイントのアイコン（UltraNavのモデルの場合は UltraNavのアイコン）を表示している場合は、アイコンをクリックして表示されるメニューから、トラックポイントのプロパティ （UltraNavのモデルの場合は詳細設定）を選択すると「マウスのプロパティ」を開くことができます。次に、トラックポイントのタブをクリックします。UltraNavのモデルの場合は、UltraNavのタブをクリックした後トラックポイントの設定ボタンをクリックします。

     

  2. スクロール機能の切り替え：の欄で、スクロール、拡大表示、オフの中からトラックポイントのセンター・ボタンの機能を選びます。

     

  3. スクロール機能や拡大表示機能の詳細な設定を行うには、設定ボタンをクリックします。「センターボタンの設定」ウィンドウが開きます。

     

  4. スクロール機能のタイプを選択します。

     

  5. 拡大表示ウィンドウの拡大率とサイズを選択します。「任意」のサイズを選択した場合は、拡大表示ウィンドウの幅と高さをピクセル単位で入力します。

     

  6. 「センターボタンの設定」ウィンドウのOKボタンをクリックします。

     

  7. 新しい設定を保存するには「トラックポイントのプロパティ」パネルの適用ボタンをクリックします。
